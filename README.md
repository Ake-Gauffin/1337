
# 1337

The command that shall not be passed. For the culture.

A tiny Linux command that prints **ELITE** in ASCII-art.


# Requirements
- `toilet`


## Install

**MacOS (Homebrew)**

```
brew install toilet 
git clone https://github.com/Ake-Gauffin/1337.git
cd 1337
chmod +x 1337
sudo cp 1337 /usr/local/bin/
cd
```

**Linux (apt)**

```
sudo apt install toilet
git clone https://github.com/Ake-Gauffin/1337.git
cd 1337
chmod +x 1337
sudo cp 1337 /usr/local/bin/
cd
```

## Usage

run command ```1337```
